# React Interview Prep

A flashcard app for studying interview questions, with two study modes.

**Live demo:** https://danielzabek11.github.io/dev-quizz

## Features

- **Browse mode** — flip through all questions at your own pace
- **Spaced Repetition mode** — SM-2 inspired algorithm that prioritizes cards you struggle with; rate each answer (Again / Hard / Good / Easy) to schedule its next appearance
- Session timer and completion screen with a per-card review chart

## Getting started

```bash
npm install
npm start
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
